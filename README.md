# portfolio-sakshi-dangi

Personal portfolio website of **Sakshi Dangi** — AI & Machine Learning Engineer, B.Tech CSE (AI/ML) student at Acropolis Institute of Technology and Research, Indore.

Built with **Next.js 16**, **TypeScript**, **Tailwind CSS v4**, and **Framer Motion**. Deployed on Vercel.

---


## About

This portfolio showcases my projects, technical skills, and background in AI/ML engineering and backend development. It features:

- **Hero section** with profile, stats, and resume download
- **Projects section** highlighting AI-powered full-stack apps
- **Skills section** with tech stack overview
- **Achievements** and highlights
- **Contact form** with direct links to GitHub, LinkedIn, and email

## Running Locally

```bash
git clone https://github.com/sakshidangi2006/portfolio-sakshi-dangi.git
cd portfolio-sakshi-dangi
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
